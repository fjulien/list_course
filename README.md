## Installation

### Front initialization 

- 1 . Go to the folder.

- 2 . Npm dependencies installation :

`$ npm install`

- 3 . Start the front project :

`$ npm start`

- 4 . And to finish, in your navigator,  go to *http://localhost:3000/*  to see the web project.
